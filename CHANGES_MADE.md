# 🎨 Внесенные изменения

## Дата: 31 марта 2026

### ✅ Выполненные задачи

#### 1. **Исправление бургер-меню для мобильных устройств**

**Проблема:** Бургер-меню размывалось при выезде в мобильном режиме из-за свойства `backdrop-filter: blur(8px)`.

**Решение:**
- Удалено размытие фона (`backdrop-filter: blur`)
- Увеличена непрозрачность затемнения с 0.4 до 0.6 для лучшей читаемости
- Упрощены CSS-свойства для повышения производительности
- Добавлено сглаживание шрифтов для четкости текста

**Измененные файлы:**
- `static/css/components.css` - стили бургер-меню

**Ключевые изменения:**
```css
/* Было:**
.burger-menu-overlay {
    background: rgba(25, 25, 30, 0.4);
    backdrop-filter: blur(8px);
    -webkit-backdrop-filter: blur(8px);
}

/* Стало:**
.burger-menu-overlay {
    background: rgba(25, 25, 30, 0.6); /* Без размытия */
}

.burger-menu {
    -webkit-font-smoothing: antialiased;
    -moz-osx-font-smoothing: grayscale;
}
```

#### 2. **Дизайн модального окна "Заказать звонок"**

**Задача:** Создать красивую, современную форму обратного звонка в модальном окне.

**Реализация:**

##### HTML Структура (`templates/partials/footer.html`):
- Двухколоночный макет:
  - **Левая часть:** Заголовок, описание, преимущества с иконками
  - **Правая часть:** Форма с полями (Имя, Телефон, Email, Выбор класса)
- Добавлены SVG-иконки для преимуществ
- Поля для UTM-меток
- Чекбокс согласия на обработку данных

##### CSS Стили (`static/css/components.css`):
- Современный градиентный фон
- Анимация появления (slide-in эффект)
- Стильные поля ввода с подсветкой при фокусе
- Градиентная кнопка отправки
- Полная адаптивность под мобильные устройства

**Особенности дизайна:**
- **Цвета:** #5773bc (синий), #ec4899 (розовый градиент)
- **Шрифты:** Inter для форм, Druk Cyr для заголовков
- **Эффекты:** 
  - Плавная анимация при открытии (0.4s)
  - Тень при наведении на кнопку
  - Подсветка полей при фокусе
  - Поворот кнопки закрытия при наведении

##### JavaScript Функционал (`templates/partials/header.html`):
- Кнопка "Заказать звонок" теперь открывает модальное окно
- Используется функция `openPopup('callback')`

```javascript

// или клик по фону/крестику
```
