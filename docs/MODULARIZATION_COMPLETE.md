## 🔧 Как это работает

### Jinja2 Include

Используется механизм наследования и включения шаблонов Jinja2:

```html
<!-- Базовый шаблон -->
{% extends "base.html" %}

<!-- Подключение секции -->
{% include 'sections/section_X_name.html' %}
```

При рендеринге FastAPI подставляет содержимое каждого файла в нужное место.

## 📝 Примеры использования

### Добавить новую секцию

```bash
# 1. Создайте файл
touch templates/sections/section_14_new.html

# 2. Напишите HTML код
# ... содержание секции ...

# 3. Добавьте в index.html
{% include 'sections/section_14_new.html' %}
```

### Изменить порядок секций

В файле `templates/index.html` просто поменяйте местами строки:

```html
<!-- Было -->
{% include 'sections/section_3_subjects.html' %}
{% include 'sections/section_4_benefits.html' %}

<!-- Стало -->
{% include 'sections/section_4_benefits.html' %}
{% include 'sections/section_3_subjects.html' %}
```

### Отключить секцию

Закомментируйте или удалите строку в `index.html`:

```html
<!-- {% include 'sections/section_12_license.html' %} -->
```

### Создать A/B версию

```bash
# Создайте две версии одной секции
cp templates/sections/section_8_reviews.html \
   templates/sections/section_8_reviews_A.html

cp templates/sections/section_8_reviews.html \
   templates/sections/section_8_reviews_B.html
```

Затем подключайте нужную версию в зависимости от условий.
