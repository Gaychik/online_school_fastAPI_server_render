# ЕГЭЛЭНД - Лендинг онлайн-школы на FastAPI

Полная копия оригинального лендинга, переписанная с использованием FastAPI и серверного рендеринга Jinja2.

## 📋 Описание проекта

Этот проект представляет собой современный лендинг онлайн-школы для подготовки к ЕГЭ и ОГЭ со следующими особенностями:

- **Серверный рендеринг** на FastAPI + Jinja2
- **100% сохранение стилистики** оригинального сайта
- **Адаптивный дизайн** под все устройства
- **Интерактивные элементы**: слайдеры, аккордеоны, таймеры, модальные окна
- **Формы обратной связи** с валидацией
- **Модульная структура** шаблонов

## 🛠 Установка и запуск

### Шаг 1: Установка зависимостей

```bash
# Создайте виртуальное окружение
python -m venv venv

# Активируйте его (Windows)
venv\Scripts\activate

# Или (Linux/Mac)
source venv/bin/activate

# Установите зависимости
pip install -r requirements.txt
```

### Шаг 2: Запуск сервера

```bash
# Запуск приложения
python main.py
```

Или через uvicorn напрямую:

```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### Шаг 3: Открытие в браузере

Перейдите по адресу: **http://localhost:8000**

## 📡 API Endpoints

### POST `/api/submit-form`
Обработка форм заявки.

**Пример запроса:**
```json
{
  "name": "Иван Иванов",
  "phone": "+7 (999) 000-00-00",
  "email": "ivan@example.com",
  "telegram": "@username"
}
```

**Ответ:**
```json
{
  "success": true,
  "message": "Заявка успешно отправлена"
}
```

### GET `/thank-you`
Страница благодарности после успешной отправки формы.

## 💡 Советы по разработке

### Добавление новой секции:

1. Создайте HTML в `templates/sections/new_section.html`
2. Добавьте стили в `static/css/sections.css`
3. Подключите в `index.html`: `{% include 'sections/new_section.html' %}`

### Изменение цветов:

Откройте соответствующий CSS файл и найдите переменную цвета.

### Добавление формы:

1. Обновите модель `ContactForm` в `main.py`
2. Добавьте обработку в endpoint `/api/submit-form`
3. Обновите JavaScript для AJAX отправки
