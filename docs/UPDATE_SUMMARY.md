### 4. **UTM-метки** 📊

#### Реализация:
- Автоматический захват UTM-параметров из URL
- Сохранение в скрытых полях формы
- Передача на сервер при отправке формы
- Логирование в консоль для отладки

#### Поддерживаемые метки:
- `utm_source` - источник трафика
- `utm_medium` - тип трафика
- `utm_campaign` - название кампании
- `utm_term` - ключевое слово
- `utm_content` - содержание объявления

#### Пример URL с UTM:
```
http://localhost:8000/?utm_source=yandex&utm_medium=cpc&utm_campaign=spring2024
```

#### Файл: `main.py`

##### Изменения:

## 🚀 Как использовать

### 1. Открыть модальное окно:
```javascript
// Автоматически работает для всех кнопок с классами:
// .package-btn, .package__btn, .mc_button.to-popup, .to-popup
// Или вручную:
openPopup('callback')
```

### 2. Закрыть модальное окно:
```javascript
closePopup('callback')
```

### 3. Навигация по секциям:
```html
<!-- Просто используйте href с ID секции -->
<a href="#prices">Тарифы</a>
```

### 4. UTM-метки:
```bash
# Просто добавьте параметры к URL
http://localhost:8000/?utm_source=yandex&utm_medium=cpc
```
