## 💡 Как использовать

### Подключить секцию:

```html

```

### Изменить порядок:

Поменяйте местами строки в `index.html`

### Добавить новую:

1. Создайте `templates/sections/section_14_NEW.html`
2. Добавьте `{% include 'sections/section_14_NEW.html' %}` в index.html
