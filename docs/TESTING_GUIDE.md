### ✅ 4. UTM-метки

**Где проверять:**
- Откройте консоль сервера (терминал, где запущен python)
- Или смотрите логи FastAPI

**Как тестировать:**

**Шаг 1: Добавьте UTM-метки к URL**
```
http://localhost:8000/?utm_source=yandex&utm_medium=cpc&utm_campaign=spring2024&utm_term=math&utm_content=test123
```

**Шаг 2: Отправьте форму**
- Откройте модальное окно
- Заполните все поля
- Нажмите "Записаться"

**Шаг 3: Проверьте логи сервера**
В терминале должны быть строки:
```
New form submission:
  Name: ...
  Phone: ...
  Class: ...
  UTM Source: yandex
  UTM Medium: cpc
  UTM Campaign: spring2024
  UTM Term: math
  UTM Content: test123
```

**Автоматический захват:**
- [ ] UTM-метки из URL автоматически попадают в скрытые поля формы
- [ ] При отправке метки передаются на сервер
- [ ] Метки логируются в консоль

## 🐛 Отлов багов

### Если бургер-меню не работает:

1. Откройте консоль браузера (F12)
2. Посмотрите ошибки JavaScript
3. Проверьте, что элемент `.burger-menu` существует в DOM
4. Проверьте, что `.mobile-toggle` существует

### Если модальное окно не открывается:

1. Проверьте класс кнопки (должен быть `.to-popup` или `.package__btn`)
2. Убедитесь, что JavaScript загружен
3. Проверьте, что `openPopup('callback')` вызывается

### Если навигация не работает:

1. Проверьте ID секций через инспектор элементов
2. Убедитесь, что href начинается с `#`
3. Проверьте, что smooth scroll работает в браузере

### Если UTM не собираются:

1. Проверьте, что URL содержит параметры (`?utm_source=...`)
2. Убедитесь, что hidden поля есть в форме
3. Смотрите логи сервера при отправке
